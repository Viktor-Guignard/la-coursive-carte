Réglages — sauvegarde cloud

Les versions sont lisibles partout sans réglage. Pour enregistrer depuis cet ordinateur, collez une fois votre jeton GitHub.

  1. Ouvrez github.com/settings/personal-access-tokens/new (connecté à votre compte GitHub).
  2. Token name : « carte coursive » — Expiration : 1 an.
  3. Repository access : Only select repositories → choisissez la-coursive-carte.
  4. Permissions → Repository permissions → Contents : Read and write.
  5. Bouton Generate token, copiez le jeton (commence par github_pat_…) et collez-le ci-dessous.
Ce jeton ne donne accès qu'au repo de la carte, rien d'autre. À coller une fois par ordinateur (PC du resto, Mac, etc.).

Versions enregistrées

Chaque enregistrement crée une nouvelle version horodatée, sans jamais écraser les précédentes. La plus récente se charge automatiquement à l'ouverture du site.
